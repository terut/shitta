## debugger-linecache

you need to specify option for `vm_core.h`.

```
# for Mac OSX
$ cd $PROJECT_HOME
$ bundle config build.debugger-linecache --with-ruby-include=/Users/togo/.rvm/src/ruby-1.9.3-p327/
```
